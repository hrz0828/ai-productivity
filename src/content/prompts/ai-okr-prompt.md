## 适用场景

适合季度规划、团队目标制定、个人成长计划和项目阶段目标拆解。

## Prompt 正文

```text
你是一名目标管理顾问。请根据我的目标想法，帮我生成一份 OKR 初稿。

要求：
- Objective 要清晰、有方向感，不要写成任务
- Key Results 要可衡量，尽量包含数字或明确完成标准
- 每个 KR 下给出 2-3 个行动建议
- 标记可能的风险和依赖

目标想法：
[粘贴内容]
```

## 输入示例

```text
我想在下个季度把 AI 内容站做起来，提升搜索流量，并准备 AdSense 审核。
```

## 输出示例

```markdown
## Objective
建立一个结构完整、内容充实、具备商业化基础的 AI 生产力内容站。

## Key Results
1. 发布不少于 50 篇原创内容，覆盖 Prompt、Workflow、Tool 和 Guide。
2. 完成 Google Search Console 配置，并让核心页面进入索引。
3. 完成 AdSense 审核前准备，包括 ads.txt、隐私政策和内容质量优化。
4. 每周至少复盘一次搜索表现和内容更新计划。

## 风险
- 内容更新速度不足
- 新站收录周期较长
- 部分文章深度不够
```

## 检查重点

OKR 不应只是任务清单。KR 应该能回答“如何判断目标达成”。
