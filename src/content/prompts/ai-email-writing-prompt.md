## 适用场景

当你需要写项目同步、催办、合作沟通、会议邀请或问题反馈邮件时，可以用这个 Prompt 快速生成初稿。

## Prompt 正文

```text
你是一名专业的职场沟通写作助理。请根据以下信息帮我写一封工作邮件。

请先理解邮件目标，再生成邮件正文。

输入信息：
- 收件人：
- 邮件目标：
- 背景信息：
- 需要对方做什么：
- 截止时间：
- 语气：正式 / 友好 / 简洁 / 委婉

输出要求：
1. 邮件标题
2. 邮件正文
3. 如果适合，补充一个更简洁版本
4. 不要夸大，不要加入未提供的承诺
```

## 输入示例

```text
收件人：研发负责人
邮件目标：确认搜索功能上线时间
背景信息：页面已经开发完成，等待最后构建验证
需要对方做什么：确认是否可以本周五上线
截止时间：今天下班前回复
语气：简洁、专业
```

## 输出示例

```text
标题：确认搜索功能本周五上线安排

你好，

搜索功能页面目前已经完成开发，并已进入最后构建验证阶段。想和你确认一下，该功能是否可以按计划在本周五上线。

如果上线时间有调整，麻烦今天下班前同步一下原因和新的预计时间，方便我更新项目计划。

谢谢。
```

## 使用建议

邮件 Prompt 最重要的是明确“希望对方做什么”。如果只提供背景，AI 可能写成说明文，而不是行动导向的邮件。
