## 这个工作流解决什么问题

内容站持续更新最难的是稳定选题和排期。内容日历工作流可以把“想写什么”变成“什么时候写、写给谁、为什么写”。

## 输入信息

- 内容主题
- 目标用户
- 核心栏目
- 目标关键词
- 更新频率
- 商业化目标

## 工作流步骤

1. 生成选题池。
2. 按搜索意图分类。
3. 标记内容类型：Prompt、Guide、Workflow、Tool。
4. 设定发布时间。
5. 规划内链。
6. 定义复盘指标。

## 输出示例

| 日期 | 标题 | 栏目 | 关键词 | 目标 |
|---|---|---|---|---|
| 第 1 周 | AI 写周报 Prompt | Prompt | AI 周报 Prompt | 获取长尾流量 |
| 第 2 周 | Claude Code 入门 | Tool | Claude Code 教程 | 工具教程入口 |
| 第 3 周 | 用 AI 写 PRD | Workflow | AI PRD | 产品经理人群 |

## 使用建议

不要让 AI 一次生成全年计划。先做 4 周内容日历，每周根据 Search Console 数据调整。
