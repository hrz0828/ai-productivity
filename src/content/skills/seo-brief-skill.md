## 这个 Skill 解决什么问题

SEO 文章质量不稳定，往往是因为写作前没有清晰简报。SEO Brief Skill 用于在写作前明确目标关键词、读者需求和文章结构。

## 输入要求

- 目标关键词
- 目标读者
- 业务目标
- 参考文章
- 必须覆盖的问题

## 执行步骤

1. 分析搜索意图。
2. 拆分用户问题。
3. 设计文章结构。
4. 生成标题和描述建议。
5. 给出 FAQ 和内链建议。

## 输出格式

```markdown
## 目标关键词
## 搜索意图
## 读者问题
## 推荐标题
## 文章大纲
## FAQ
## 内链建议
```

## 注意事项

Brief 不是最终文章。写作时仍然需要真实经验、案例和清晰表达，避免只堆关键词。

## 使用案例：为“Claude Code 入门”生成写作简报

### 场景

你准备写一篇 Claude Code 入门教程，希望先明确读者搜索意图和文章结构。

### 输入示例

```text
目标关键词：Claude Code 入门
目标读者：第一次使用 Claude Code 的开发者
业务目标：获得搜索流量，并引导阅读 AI 编程工作流内容
```

### 输出示例

SEO Brief 应包括搜索意图、读者问题、推荐标题、文章结构、FAQ、内链建议和需要补充的实操截图。

### 使用建议

生成 Brief 后，不要直接让 AI 写完全文。先补充自己的实际操作经验，再逐节生成内容，文章会更可信。
