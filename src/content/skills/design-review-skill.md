## 这个 Skill 解决什么问题

设计评审很容易变成主观评价：

- “这个页面不够高级。”
- “这个按钮不够突出。”
- “这个排版感觉有点乱。”

这些反馈很难执行。设计评审 Skill 的目标是把主观感受转成结构化检查项，让 AI 按固定维度输出问题、原因和修改建议。

它适合评审：

- 落地页
- 产品首页
- 功能页面
- 表单页面
- 空状态 / 错误状态
- 活动页
- App 或 Web 页面截图

## 输入要求

使用前最好准备：

- 页面目标：这个页面希望用户做什么
- 目标用户：新用户、老用户、管理者、开发者等
- 页面截图或 Figma 链接说明
- 关键操作：注册、购买、提交、下载、联系销售等
- 当前担心的问题
- 设计阶段：草稿、评审前、上线前、改版后

如果不能上传图片，也可以用文字描述页面结构。

## Skill Prompt

```text
你是一名资深 UX 设计评审专家，擅长从目标、信息层级、文案、交互和可用性角度评审页面。

请根据我提供的页面目标、用户背景和设计说明，做一次结构化设计评审。

输入信息：
页面类型：
页面目标：
目标用户：
关键操作：
页面结构 / 截图说明：
当前担心的问题：
设计阶段：

请按以下维度评审：
1. 页面目标是否清晰
2. 信息层级是否合理
3. 主 CTA 是否突出
4. 文案是否准确、具体、可理解
5. 用户路径是否顺畅
6. 空状态、错误状态、加载状态是否考虑
7. 移动端和响应式风险
8. 可访问性问题
9. 视觉一致性和组件复用
10. 需要通过用户测试验证的问题

输出格式：
## 总体判断
## 主要问题
## 次要问题
## 具体修改建议
## 需要补充的信息
## 上线前检查清单

要求：
- 不要只说“好看 / 不好看”
- 每个问题都说明影响和修改建议
- 区分高优先级和低优先级
- 如果缺少截图或信息，请先说明评审限制
```

## 示例输入

```text
页面类型：AI 内容站首页
页面目标：让新用户理解网站价值，并点击进入 Prompt 或 Workflow
目标用户：想用 AI 提升办公效率的中文知识工作者
关键操作：搜索内容、查看工作流、理解 Prompt / Skill / Agent / Workflow
页面结构 / 截图说明：顶部是 Hero 标题和搜索框，右侧有“本站适合解决”列表，下方是热门主题、精选内容和最新文章
当前担心的问题：首页内容很多，但用户可能不知道从哪里点
设计阶段：改版前评审
```

## 预期输出方向

AI 应该指出：

- “本站适合解决”如果只是文本，不利于转化，建议改为可点击场景卡片。
- Prompt、Skill、Agent、Workflow 对新手有门槛，建议增加简短解释。
- 热门主题如果内容过少，可能让用户觉得网站未完成。
- 最新文章和工具教程如果重复，会浪费首页展示空间。

## 评审结果模板

```markdown
## 总体判断
页面内容较丰富，但当前信息架构更像内容陈列，而不是任务导向入口。建议优先强化用户场景入口和概念解释。

## 主要问题
| 问题 | 影响 | 优先级 | 修改建议 |
| --- | --- | --- | --- |
| 场景列表不可点击 | 用户看到问题后无法直接进入解决方案 | 高 | 改成场景卡片，并链接到对应 Prompt / Workflow |
| 核心概念缺少解释 | 新手不理解 Prompt、Skill、Agent、Workflow 区别 | 高 | 增加“从简单到复杂”的说明模块 |

## 次要问题
- 卡片样式较统一，内容类型辨识度不足。
- 分类入口较多，建议优先展示内容充足的主题。

## 上线前检查清单
- 主要 CTA 是否明确？
- 用户是否能在 10 秒内理解页面价值？
- 每个场景入口是否有对应链接？
- 移动端首屏是否过长？
```

## 可组合的其他 Skill

这个 Skill 可以和以下内容组合使用：

- SEO Brief Skill：检查页面是否符合搜索意图。
- 文案改写 Skill：优化页面标题、说明和 CTA。
- SOP 生成 Skill：把设计评审流程沉淀为团队规范。
- 会议纪要 Skill：把设计评审会议整理成行动项。

## 适合搭配的工具

- Figma：提供设计稿和页面截图。
- ChatGPT / Claude：进行结构化评审和文案修改。
- Gemini：处理截图、多模态资料和长文说明。
- Notion：沉淀评审记录和设计规范。

## 使用边界

AI 设计评审不能替代：

- 真实用户测试
- 可用性测试
- 品牌视觉规范审核
- 前端实现检查
- 数据转化分析

AI 更适合在评审前帮你发现明显遗漏，或把模糊反馈整理成可执行建议。
