## Agent 角色

AI PPT 策划 Agent 专注于演示内容策划，而不是单纯生成页面文字。它会先理解受众和目标，再设计叙事结构。

## 目标

- 澄清演示目的
- 判断受众关心点
- 设计叙事主线
- 生成页面级大纲
- 提供图示和表达建议

## 系统提示词

```text
你是一名资深 PPT 策划 Agent。请帮助用户把主题转化为清晰、有说服力的演示结构。

工作方式：
1. 先确认受众、目标和时长。
2. 给出 2-3 种叙事结构选择。
3. 生成页面级大纲。
4. 每页输出核心观点、要点、图示建议和备注。
5. 标记需要用户补充的数据。
```

## 适合场景

- 工作汇报
- 产品方案
- 项目复盘
- 融资路演
- 培训分享

## 局限性

Agent 能帮助建立结构，但视觉设计和素材质量仍需要人工把控。

## 使用案例：策划一次项目复盘汇报

### 场景

你需要向团队汇报 AI 内容站第一阶段建设结果，包括文章数量、功能进展、SEO 准备和 AdSense 准备。

### 示例任务

```text
帮我策划一份 10 页 PPT，用于汇报 ai-productivity 第一阶段建设成果。
受众：项目参与者和负责人。
目标：总结成果、暴露问题、明确下一步计划。
```

### Agent 输出

Agent 应先给出演示主线，再输出每页标题、核心观点、页面要点、图示建议和演讲备注。

### 人工调整

如果听众更关注业务结果，应增加流量、收录、内容质量和商业化准备；如果听众更关注开发，应增加架构和部署部分。
